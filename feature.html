Oh My Code

Oh My Code est un générateur d’application



Pour générer une application sans développeurs ni connaissances technique, vous trouverez forcément votre bonheur. Une solution originale pour créer des pages au format html sans programmer. Attention, ces pages ne seront exploitables que si vous avez un minimum de rudiments car elles nécessitent ensuite de pouvoir compléter le code.

Le principe est donc très simple. On dessine simplement sa page. Pour faire simple, le système reconnaît des éléments comme étant du texte (une succession de lignes par exemple) des images (des cadres vides) ou encore des zones de saisie. Vous dessinez et envoyer votre dessin sur l 'api.

La reconnaissance est impressionnante, mais il faudra quelques essais avant de parvenir à maîtriser toutes les subtilités de la reconnaissance. il faudra aussi une bonne dose d’imagination pour penser un site. Bien entendu le fichier html récupéré n’est pas complet car il faudra ajouter les véritables images et le véritable texte mais c’est très pratique en particulier si vous souhaitez créer une simple page d’accueil.

Le besoin

Le processus de conception d'interface utilisateur implique une créativité qui commence par un tableau blanc où les concepteurs partagent des idées.

Une fois qu'un dessin est dessiné, il est généralement capturé sur une photo et traduit manuellement en une structure filaire HTML fonctionnelle à utiliser dans un navigateur Web.

L'idée

La vision par ordinateur est une discipline de l'intelligence artificielle qui donne à une application la capacité de voir et de comprendre ce qu'elle voit.

Nous pouvons utiliser cette technologie pour construire un système qui comprend ce qu'un concepteur a dessiné sur un tableau blanc et qui peut traduire cette compréhension en code.

De cette façon, nous pouvons générer des images filaires HTML directement à partir d'une image dessinée à la main, ce qui permet une implémentation instantanée de la conception, simplifiant ainsi le processus de création.

+ de détails

La solution

Le modèle à l'origine de ce service a été formé avec des millions d'images et permet la détection d'objets pour une grande variété de types d'objets.

Dans ce cas, nous devons créer un modèle personnalisé et le former à l'aide d'images d'éléments de conception dessinés à la main, tels qu'une zone de texte, un bouton ou une liste déroulante.

Une fois que nous pouvons identifier les objets, nous utilisons la fonctionnalité de reconnaissance de texte par ordinateur pour extraire le texte écrit à la main présent dans la conception.

En combinant ces deux informations, nous pouvons générer les extraits des différents éléments de la conception.

Nous pouvons ensuite déduire la mise en page du dessin à partir de la position des éléments identifiés et générer le code final en conséquence.
